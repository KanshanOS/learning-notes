# Git 常用操作
---
## Git 基础
### 获取 Git 仓库
- 在现有目录中初始化仓库
```sh
git init
git add .
git commit -m "初始化提交"
```
- 拉取远程仓库
```
git clone https://github.com/Yu-Hai/StudyNotes.git stydu-notes
```

> 若无 stydu-notes ，拉取下来后默认创建 StudyNotes 目录

### 查看提交历史
```sh
git log                 # 展示所有更新
git log -2              # 展示进两次更新
```

### 远程仓库
- 同步远程仓库信息
```sh
git fetch https://github.com/Yu-Hai/StudyNotes.git
```
- 推送本地数据到远程仓库
```sh
git push origin master      # git push [remote-name] [branch-name]
```

## Git 分支
### 分支切换
- 查看所有分支
```sh
git log --oneline --decorate
# 或者使用查看分支列表
git branch
```
- 切换分支
```sh
git checkout test-branch
```

### 分支的创建与合并
- 创建分支
```sh
git checkout -b new-branch
# 等价于
git branch new-branch
git checkout new-branch
```

- 合并分支
```sh
git checkout master
git merge new-branch
```

- 删除分支
```sh
git branch -d new-branch
```
